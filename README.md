# Level Regulation & System Identification
**Academic Project | Technical University of Cluj-Napoca**

## Project Overview
This project focuses on the mathematical modeling and automated control of a **two-tank serial process**. The main objective was to maintain a precise fluid level using feedback control strategies.

## Key Technical Tasks
* **Mathematical Modeling:** Derived the differential equations and transfer functions for the serial tank system.
* **System Identification:** Performed identification procedures to determine the process parameters.
* **Controller Design:** Designed and tuned **PID controllers** for optimal level maintenance and stability.
* **Simulation & Validation:** Implemented and tested the control logic using **MATLAB & Simulink**.

## Software Used
* MATLAB
* Simulink
