# Level Regulation & System Identification
**Academic Project | [cite_start]Technical University of Cluj-Napoca** 

## Project Overview
This project focuses on the mathematical modeling and automated control of a **two-tank serial process**. [cite_start]The main objective was to maintain a precise fluid level using feedback control strategies. [cite: 17]

## Key Technical Tasks
* [cite_start]**Mathematical Modeling:** Derived the differential equations and transfer functions for the serial tank system. [cite: 17]
* **System Identification:** Performed identification procedures to determine the process parameters. [cite: 17]
* [cite_start]**Controller Design:** Designed and tuned **PID controllers** for optimal level maintenance and stability. [cite: 18]
* [cite_start]**Simulation & Validation:** Implemented and tested the control logic using **MATLAB & Simulink**.
